# Claude Skills

Коллекция навыков для Claude Code от Антони Вдовиченко ([@aiwizards](https://t.me/aiwizards)).

Здесь собраны навыки, которые я делаю сам, и те, которыми пользуюсь в работе.

## Установка

Склонируйте репозиторий и скопируйте нужный навык в `.claude/skills/` вашего проекта:

```bash
git clone git@github.com:miolamio/claude-skills.git
cp -r claude-skills/en-ru-translator-adv /path/to/your/project/.claude/skills/
```

Если навык нужен во всех проектах, положите его в личную папку:

```bash
cp -r claude-skills/en-ru-translator-adv ~/.claude/skills/
```

Claude Code подхватит навык автоматически — перезапуск не требуется.

## en-ru-translator-adv

Переводчик с английского на русский. Работает в три этапа: черновой перевод, рефлексия и финальная редактура. Результат читается так, будто текст изначально написан на русском.

**Версия:** 3.2.0

**Что умеет:**

- Переводит техническую документацию, маркетинговые материалы, статьи, учебные тексты и строки интерфейса.
- Следует принципам информационного стиля Максима Ильяхова («Пиши, сокращай»).
- Вычищает кальки, канцеляризмы, стоп-слова и маркеры машинного перевода.
- Ведёт единую терминологию через глоссарий.
- Применяет правила русской типографики: кавычки-ёлочки, тире, оформление списков, буква «ё».
- Корректно обрабатывает смешанные тексты — код и команды остаются на английском, общеупотребительные термины переводятся.

**Справочные материалы (`references/`):**

- `anti-patterns.md` — каталог ошибок перевода: кальки, англицизмы, ложные друзья переводчика, номинализация, пассивный залог.
- `examples.md` — примеры переводов по типам текста: маркетинг, техдок, образование, бизнес.
- `glossary-strategy.md` — стратегия ведения глоссария и работа с терминологией.
- `informational-style.md` — правила информационного стиля по Ильяхову.
- `typography.md` — правила русской типографики: кавычки, тире, списки, числа, даты, сокращения.
